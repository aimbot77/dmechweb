[03] Why Sovereign Matters

National Security
Requires National
Infrastructure

Defense-critical decision systems cannot depend on foreign frameworks, cloud connectivity, or external data pipelines. AEGIS-X is built for air-gapped sovereignty.

Air-Gapped Operation

Zero cloud dependency. No external API calls, no telemetry, no data exfiltration surface. The system operates identically whether connected or isolated.

No Foreign ML Dependencies

The CCPL reinforcement-learning framework is indigenous. No dependency on foreign ML libraries that could be restricted, backdoored, or deprecated under geopolitical pressure.

Full Audit Trail

Every decision is logged with timestamp, sensor inputs, threat assessment, and ROE state. Complete accountability for every engagement action, from detection to response.

Deployment & Hardening

OS & User Space

Bare-metal install: Ubuntu 22.04, dedicated aegis_user with no login shell. Strict systemd hardening: NoNewPrivileges, ProtectSystem=strict, PrivateTmp.

Air-Gapped Targets

--offline flag for fully air-gapped deployment. Docker + Pi 4 ARM64 deploy targets natively supported.

Latency Profiling & Licensing

Latency profiler explicitly benchmarks Pi 4 ARM64 (simulation numbers from x86 laptops do not transfer). License validator uses HMAC signing key via /run/secrets/.
